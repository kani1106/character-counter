# Real-Time Character Counter Message Box

A simple and interactive **message box with a real-time character counter** built using HTML, CSS, and JavaScript. This project shows users the number of characters they type dynamically.

## Features

- Real-time character count while typing.
- Customizable maximum character limit.
- User-friendly and responsive interface.
- Easy to integrate into other web projects.
## Technologies Used

- HTML5
- CSS3
- JavaScript
